# 🎮 Mundo de los Videojuegos: Shooters 🎮
========================================================

## Descripción

Este proyecto consiste en una plataforma web informativa dedicada al género de los videojuegos Shooters (disparos). El objetivo principal fue desarrollar una Web utilizando únicamente HTML y CSS, aplicando conceptos de diseño dinámicos y experiencia de usuario.

## ✏️ Paso a Paso de Creación ✏️

Para armar esta página, seguí este orden de trabajo:
1. **Estructura HTML:** Creé el esqueleto del sitio usando etiquetas como `<h1>`, `<h3>`, `<ul>` y `<li>`.
2. **Contenido:** Busqué las imágenes en Google y los enlaces oficiales de cada juego.
3. **Diseño CSS:** Creé un archivo de estilos para que las listas no se vieran aburridas y se convirtieran en tarjetas (cards).
4. **Interactividad:** Añadí efectos de "hover" para que las tarjetas se muevan cuando pasas el ratón por encima.

## 👩‍💻 Detalles del Código 👩‍💻

### HTML (El Esqueleto)
1. **Listas Desordenadas ('<ul>'):** Las usé para agrupar todos los juegos.
2. **Imágenes ('<img>'):** Enlazadas directamente desde la web para mostrar el arte de cada shooter.
3. **Enlaces ('<a>'):** Configurados como botones para llevarte a la página oficial de cada juego.

### CSS (Style)
1. **CSS Grid:** Usé "display: grid" para que las tarjetas se acomoden solas dependiendo del tamaño de la pantalla.
2. **Efecto Hover:** Las tarjetas suben un poquito ('translateY') cuando pasas el mouse.
3. **Botones:** Personalicé los links para que parezcan botones redondeados con cambios de color.
